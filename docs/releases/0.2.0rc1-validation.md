## Known Non-Blockers

- `release-health` reports `warn` when run against a fresh checkout before any
  snapshot timeline exists. Run `memory --out-dir .repomori-packs --no-handoff`
  first when you want a full health pass with doctor, chain, timeline, compat,
  and contract checks.
- Historical drift logs may keep `drift-summary` in `warn` even after the latest
  release-check has strict baseline matches. Use a fresh release-candidate
  artifacts directory for candidate proof bundles.
