# RepoMori 0.2.0rc1

RepoMori `0.2.0rc1` is the first release candidate for the source-available
public `0.2.0` line.

## Highlights

- Offline `.repomori` repository packs with compressed source chunks, hashes,
  file metadata, imports, symbols, headings, summaries, and exact source
  recovery.
- Agent-facing context, capsule, handoff, diff-context, eval, and context-eval
  commands.
- Snapshot memory loop with doctor, safe prune, timeline, tamper-evident chain,
  anchors, and release-health checks.
- Dependency-free JSON-lines agent bridge and MCP stdio bridge.
- Public-release safety scan, baseline drift telemetry, compatibility contracts,
  and CI artifact bundles.
- Source-available licensing under PolyForm Noncommercial terms, with commercial
  use reserved for separate written permission.

## Candidate Scope

This is a pre-release for validation, packaging checks, documentation review,
and downstream workflow testing. It is not the final `0.2.0` release.

## Validation

Expected gates before publishing the pre-release:

- `python -m unittest discover -s tests`
- `python -m repomori release-check D:\Dev\RepoMori --baseline D:\Dev\RepoMori\.repomori-scan-baseline.json --fail-on low --json`
- `python -m repomori release-health D:\Dev\RepoMori --snapshot-dir D:\Dev\RepoMori\.repomori-packs --baseline D:\Dev\RepoMori\.repomori-scan-baseline.json --json`
- GitHub `tests` workflow green on `main`
- GitHub `release-candidate` workflow green for `0.2.0rc1`

## Notes

RepoMori remains local-only and model-free. It does not call OpenAI, Claude, or
any provider API, and it does not require API keys.
