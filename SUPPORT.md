## Community Support

For personal and non-commercial use:

- use GitHub issues for reproducible bugs, documentation gaps, and feature
  proposals;
- include the RepoMori version, Python version, operating system, command, and
  expected versus actual behavior;
- do not attach private repositories, secrets, generated `.repomori` packs, or
  handoff directories from private source.
